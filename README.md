        该仓库用于存档基于MOW II的自定义着色器系统（GSM）的（我了解到的）技术文档。
        和WZSS Shader Plus(WSP) mod的开发指南，以指导任何希望使用本mod增强画面的作者正确的使用它。
        和我在实现WSP时学习的图形学资料，以帮助任何想要在该游戏上编写着色器的爱好者推进他的工作。
